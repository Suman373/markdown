# Tables 

Tables are created using pipes and hyphens
For example, 
Syntax : 
```
| Fruit | Price |
--------|--------
| Banana | $2 |
| Peach | $2.4 |
```
Output:

| Fruit | Price |
|--------|:--------:|
| Banana | $2 |
| Peach | $2.4 |
